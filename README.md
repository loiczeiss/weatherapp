
# Weather App but a cool one...

A react app using two rest APIs to display weather intels based
on a location input.

https://loiczeiss.github.io/weatherapp/

## API Reference

https://api.open-meteo.com/v1/forecast

https://us1.locationiq.com/v1/search
## Features

- Background style changes based on weather conditions
- Search by country, city or street, mondially
- Cross platform


## Tech Stack

**Client:** React, TailwindCSS, axios, React-Select-Page-Async
